# Expressive Brand Chrome Surrounds Neutral Creative Content

Debrute applies its warm, flat, handmade collage language to Workbench chrome
and shared controls in both light and dark/night themes. Canvas regions used to
judge creative content remain functionally neutral and high-clarity. Terminal
uses a warm-neutral field plus a coordinated high-contrast ANSI palette whose
red, green, yellow, blue, magenta, and cyan roles remain distinct. This gives
the production tool a distinctive identity without tinting user work or
weakening terminal color meaning; expressive roughness belongs to presentation,
while layout geometry, terminal text, and interaction hit targets remain
precise.
